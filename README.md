# McHammerCoder
Repository for the MontiCore4-Hammer-Coder project

# Example Guide

##1. Installing Dependencies for Hammer:

**sudo apt-get install gcc scons pkg-config libglib2.0-0 glib2.0-dev**

##2. Installing Hammer (Maven)

@ /core/Hammer

**mvn clean install**

##3. Installing Monticore

@ /core/Monticore

**mvn clean install**

##4. Code Generation:

@ /examples/HTMLRed

**sh codegen.sh**

##5. Compile Generated Code:

@ /examples/HTMLRed

**sh compile.sh**

##6. Run TestTool:

@ /examples/HTMLRed

**sh run.sh**
